✨ 纯 Tailwind CSS 实现

无需自定义 CSS 和 !important

🎯 Tailwind 实现方案

直接在 HTML 中使用 Tailwind 类来设置移动端菜单的字体样式，无需额外的 CSS。

📝 使用的 Tailwind 类

  • text-base - 字体大小 16px
  • leading-[1.4] - 行间距 1.4
  • py-[10px] - 上下内边距 10px
  • px-3 - 左右内边距 12px (3 × 4px = 12px)
  • block - 块级元素

✅ 优势

  • 无需自定义 CSS
  • 无需使用 !important
  • 代码更简洁易维护
  • 完全响应式
  • Tailwind 原生支持

🧪 测试步骤

  1. 将浏览器窗口调整到移动端宽度（≤768px）
  2. 点击右上角的汉堡菜单按钮（☰）
  3. 右键点击任意菜单项 → 选择"检查"
  4. 在开发者工具中查看 Computed 标签页
  5. 验证：font-size: 16px, line-height: 22.4px, padding: 10px 12px

📊 预期结果

font-size: 16px ✓
line-height: 22.4px (16px × 1.4) ✓
padding-top: 10px ✓
padding-right: 12px ✓
padding-bottom: 10px ✓
padding-left: 12px ✓
