🔧 Past Conferences 日期闪烁修复

问题分析

🐛 发现的问题：

Past Conferences 卡片左上方的日期文字在页面加载时出现闪烁。

根本原因：

  1. toLocaleDateString() 的异步行为
    • 浏览器的 toLocaleDateString() 方法可能有延迟渲染
    • 不同浏览器的实现可能导致文字重排
    • locale 数据加载可能导致闪烁
  2. 多次 DOM 更新
    • Loading 状态 → API 数据加载 → 内容替换
    • 每次 innerHTML 更新都会触发重新渲染
  3. CSS 过渡效果
    • Tailwind 的 transition 类可能导致文字抖动

修复方案

❌ 修复前

const startMonth = startDate.toLocaleDateString(locale, { month: 'short' });

使用浏览器 API，可能有延迟

✅ 修复后

const startMonth = getMonthAbbr(startDate.getMonth());

使用静态数组，立即返回

技术细节

新增的 getMonthAbbr() 函数：

function getMonthAbbr(monthIndex) {
    const months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 
                    'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'];
    return months[monthIndex] || 'Jan';
}
优势：
  • ✅ 同步执行，无延迟
  • ✅ 不依赖浏览器 locale 数据
  • ✅ 跨浏览器一致性
  • ✅ 性能更好
