🔍 故障排除

如果样式仍然丢失：

1. 检查main.css是否加载
   F12 → Network → 刷新 → 查找main.css

2. 验证文件存在
   docker exec giip-frontend ls -la /usr/share/nginx/html/css/

3. 查看Nginx日志
   docker logs giip-frontend

4. 尝试不同浏览器
   Chrome隐私模式 / Firefox私密窗口
