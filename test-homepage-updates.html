📝 技术细节

Past Conferences 网格布局

/* HTML 结构 */ <div class="conferences-slider grid grid-cols-1 md:grid-cols-3 gap-6"> <!-- 会议卡片 --> </div> /* 响应式断点 */ - 移动端 (< 768px): 1 列 - 平板端 (768px - 1023px): 2-3 列 - 桌面端 (≥ 1024px): 3 列

Learn More 链接修复

/* 之前：整个卡片都有 onclick，导致链接无法点击 */ <article onclick="..."> <a href="...">Learn More</a> </article> /* 现在：只在图片和标题上有 onclick，链接独立 */ <article> <div onclick="...">图片</div> <h3 onclick="...">标题</h3> <a href="..." onclick="event.stopPropagation();">Learn More</a> </article>

Event Card 最小高度

.event-card { min-height: 140px; /* 确保卡片有足够的高度 */ display: flex; align-items: center; /* ... 其他样式 ... */ }
