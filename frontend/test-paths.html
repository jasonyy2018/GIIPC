5. 解决方案

如果 Logo 不显示：

  • 检查哪个图片路径显示为绿色边框
  • 使用成功的路径格式更新所有页面
  • 确保从 frontend 目录打开页面

如果页面链接 404：

  • 确保从 frontend/index.html 打开
  • 不要从根目录打开
  • 使用本地服务器: python -m http.server 8000
