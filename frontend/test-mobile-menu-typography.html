Mobile Menu Typography Test

Typography Updates - Archibucks Reference

The mobile menu has been updated to match Archibucks design specifications:

  • Menu Font Size: 16px (Archibucks standard for mobile menu)
  • Menu Button Size: 20px (1.25x menu text, proportional)
  • Line Height: 1.6 (comfortable reading setting)
  • Letter Spacing: 0.034em (~0.544px)
  • Padding: 12px menu items, 8px 12px button
  • Font Weight: Regular (400) for clean readability
  • Font Family: System fonts (-apple-system, BlinkMacSystemFont, etc.)
  • Item Spacing: 12px margin-bottom

Testing Instructions

  1. Click the hamburger menu icon (☰) in the top right corner
  2. Observe the mobile menu slide in from the left
  3. Check the font size and spacing of menu items
  4. Verify the letter spacing creates better readability
  5. Test the Login and Register buttons at the bottom
  6. Click outside the menu or the overlay to close it
